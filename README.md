## 🎯 سیستم ما در چه زمینه‌هایی می‌تواند کمک کند؟

### ۱. برنامه‌نویسی و توسعه نرم‌افزار
- مقایسه فریم‌ورک‌ها و زبان‌ها (مثلاً React در مقابل Vue)
- عیب‌یابی کد و پیشنهاد راه‌حل
- ارائه ساختار پروژه برای اپلیکیشن‌های مختلف
- توضیح الگوریتم‌ها و مفاهیم پیچیده
> **ستاره‌های این حوزه:** DeepSeek-R1 (تحلیل فنی)، GPT-4o mini (راهنمایی گام‌به‌گام)

### ۲. امنیت سایبری و هک اخلاقی
- طراحی مسیر یادگیری امنیت
- توضیح انواع حملات (SQL Injection, XSS) و راه‌های مقابله
- معرفی ابزارهای تخصصی (Burp Suite, Metasploit)
- تحلیل پروتکل‌های امنیتی و گواهینامه‌های معتبر (CEH, OSCP)
> **ستاره‌های این حوزه:** DeepSeek-R1 (دقت فنی بالا)، Cohere Command R (توضیحات روان)

### ۳. تحلیل داده و هوش مصنوعی
- مقایسه الگوریتم‌های یادگیری ماشین
- توضیح مفاهیم آماری و کاربرد آن‌ها
- ارائه راهنما برای پاک‌سازی و پیش‌پردازش داده‌ها
- نقشه راه یادگیری برای دانشمند داده شدن
> **ستاره‌های این حوزه:** Mistral Small (دیدگاه‌های خلاقانه)، GPT-4o mini (ساختار آموزشی)

### ۴. تولید محتوا و نویسندگی خلاق
- داستان‌نویسی کوتاه و بلند با موضوعات خلاقانه
- نوشتن مقالات وبلاگی و پست‌های شبکه‌های اجتماعی
- بازنویسی و بهبود متون موجود
- ارائه ایده‌های خلاقانه برای پروژه‌های هنری
> **ستاره‌های این حوزه:** Mistral Small + Cohere Command R (خلاقیت و روان‌نویسی)

### ۵. آموزش و یادگیری
- طراحی برنامه درسی برای هر موضوع علمی یا فنی
- خلاصه‌سازی مفاهیم پیچیده به زبان ساده
- پاسخ به سوالات درسی (ریاضی، علوم، ادبیات)
- معرفی بهترین منابع (کتاب، دوره، وب‌سایت)
> **ستاره‌های این حوزه:** GPT-4o mini (ساختاربندی)، DeepSeek-R1 (عمق مفاهیم)

### ۶. تحلیل منطقی و فلسفی
- بررسی استدلال‌های منطقی و یافتن مغالطه‌ها
- تحلیل مسائل فلسفی و اخلاقی (مثل اخلاق در هوش مصنوعی)
- مقایسه مکاتب فکری مختلف
- کمک به تصمیم‌گیری با سنجیدن مزایا و معایب
> **ستاره‌های این حوزه:** DeepSeek-R1 (استدلال قوی)، Mistral Small (عمق فلسفی)

### ۷. حل مسئله و ایده‌پردازی عمومی
- ارائه راه‌حل برای چالش‌های روزمره
- ایده‌پردازی برای پروژه‌های جدید (استارتاپی، هنری، فنی)
- تحلیل روندهای بازار و پیش‌بینی آینده
> **ستاره‌های این حوزه:** ترکیب قدرت هر ۴ متخصص + جمع‌بندی قاضی 🏆
