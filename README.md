## 💡 <b>چطور کار می‌کند؟</b>

۱. کاربر یک **Issue** جدید در مخزن باز می‌کند.  
۲. ربات (GitHub Actions) سوال را دریافت می‌کند.  
۳. سوال به یک یا چند مدل هوش مصنوعی (مثلاً GPT-4o, Llama, Phi) فرستاده می‌شود.  
۴. پاسخ نهایی تولید شده و به‌عنوان کامنت زیر Issue ثبت می‌شود.
